# 🏥 AtlasTriage AI
**AI-Powered Breast Cancer Screening for Morocco**

## 🌟 Overview
A machine learning tool designed to help nurses in rural Morocco (Atlas Mountains) prioritize high-risk biopsy cases using 6 key cellular features.

## 🌍 Features
* **Trilingual Support:** Arabic, French, and English interfaces.
* **High Precision:** Optimized for "Recall" to ensure no malignant cases are missed.
* **Offline Ready:** Designed for low-resource environments.

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Scikit-Learn, Pandas, NumPy, Joblib
* **Model:** Logistic Regression (Standardized)
